Roteiro para o uso do React:

    Instalação do Node Js:

  1. Baixar o Node Js aqui
  2. Descompactar o tarbal: # tar -xzf arquivo.tar.gz
  3. Na pasta localize o arquivo configure e execute: # ./configure
  4. Na sequencia: # make && make install
  5. No VsCode acesse o menu View (Ver) >> Terminal
  6. Visualize a versão do Node no terminal: $ node -v
  7. Visualize a versão do npm no terminal: $ npm -v
  8. Inicie um servidor local com o npx na pasta do projeto: $ npx -serve
  9. Se tudo tiver correto acesse aqui

Outra forma seria usando o VsCode

  • Instale a extensão "Live Server"
  • Abra a pasta do seu projeto
  • Dentro do html clique com o botão direito e selecione "Open with Live Server"
  • Se tudo tiver correto acesse aqui
