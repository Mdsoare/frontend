Roteiro para o uso do React:


    Instalação do Node Js:

  • 1º) Baixar o Node Js aqui
  • 2º) Descompactar o tarbal: # tar -xzf arquivo.tar.gz
  • 3º) Na pasta localize o arquivo configure e execute: # ./configure
  • 4º) Na sequencia: make && make install
  • 5º) No VsCode acesse o menu View (ver) >> terminal
  • 6º) Visualize a versão do Node no terminal: $ node -v
  • 7º) Visualize a versão do npm no terminal: $ npm -v
  • 8º) Inicie um servidor local com o npx na pasta do projeto: $ npx -serve
  • 9º) Se tudo tiver correto acesse: aqui
